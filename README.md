# Weather Diary ☀️🌧️

**Author:** Dash Gladkih

## Description
A desktop GUI application for tracking daily weather.  
You can add records with date, temperature, description, and precipitation info.  
Filter entries by date or minimum temperature.  
All data is saved/loaded from a JSON file.

## Features
- Add new weather records with validation
- View records in a sortable table
- Filter by exact date or temperature > value
- Save data to `weather_data.json`
- Load data from `weather_data.json`
- Simple and intuitive interface

## Requirements
- Python 3.7+
- No external libraries (uses only `tkinter`, `json`, `datetime`)
